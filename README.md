# Sales Performance Dashboard

## Overview
An interactive Sales Performance Dashboard built using HTML, CSS, JavaScript, and JSON data.

## Features
- Revenue, Profit, Units Sold, and Average Order Value
- Sales & Profit Trend Chart
- Revenue by Segment Chart
- Upload JSON data for visualization
- Responsive dashboard design

## Technologies Used
- HTML
- CSS
- JavaScript
- JSON

## How to Run
1. Download the project.
2. Open index.html in your browser.
3. Upload the JSON file (if required) to view the dashboard.
